## Project Overview
This is a demo prototype of an alternative underwriting engine
designed for gig economy workers using behavioral and transaction features.

## Features Used
- Average Monthly Inflow
- Income Volatility Score
- Seasonality Score
- Geo Stability Score
- Device Consistency Score

## Model Used
Random Forest Classifier

## How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python shadow_income_model.py

## Business Concept
This model predicts whether a gig worker is creditworthy
based on alternative income stability metrics instead of
traditional salary slips.

Future Scope:
- API deployment
- Real-time scoring
- Integration with banking core systems
